## Štruktúra XML
### Tag - značka (Tag)
Otváracia značka (Opening tag): <>	
Uzatvaracia značka (Closing tag): </>

Rozdelenie značiek:
Párové (pair): <dokument> </dokument>
Nepárové (Unpair)> <dokument />

### Element
<sprava>Ahoj občan </sprava>
